# Environment Variables

- `MONGODB_URI`, the mongodb database uri
- `PORT` the http server port. By default is `3000`

# Installation

```
git clone 
cd nodejs-imagshare
npm install
npm run build
npm start
```

# Installation with docker-compose (Recommended)

```
docker-compose up
```
